Instructions to run code:

1. PASCAL Dataset folder VOCtrainval_06-Nov-2007 has to be appended to the existing folder for the code to access image data set.
2. Run pip to install all the required packages mentioned in the requirements.txt
3. Run the Main.py file to run the projcet code. Args must be passed to the file.
   To run the code enter py Main.py --k {number of candidates} --query {image filename}
   Ex: py Main.py --k 100 query 000007
